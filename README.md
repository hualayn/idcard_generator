# idcard-generator

> 使用 nodejs 生成身份证
> 
> 本工具仅供研究使用，请勿用于违法用途，否则后果自负

## 使用

```bash
git clone https://github.com/hualayn/idcard_generator.git  # 克隆项目到本地

npm install  # 安装依赖

node test  # 测试生成图片（在当前文件夹下，生成一个output.png图片）

node gen 3 # 生成 3 张图片（位于id_pics_out文件夹内）; 默认5张，可在`gen.js`修改
```
