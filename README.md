# idcard-generator

> 使用 nodejs 生成身份证
> 
> 本工具仅供研究使用，请勿用于违法用途，否则后果自负

## 使用
1. git clone 项目到本地

2. npm install 安装依赖

3. 测试：命令行输入 node test

4. 生成图片： node gen <number> // 生成<number>张图片, 默认5张，可修改
